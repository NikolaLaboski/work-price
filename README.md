# React Pricing SPA (MK)

Single Page Application со React + Vite и чист CSS.

## Како да стартуваш локално
```bash
cd react-pricing-spa
npm install
npm run dev
```
Отвори ја адресата што ќе се појави во терминалот (обично http://localhost:5173).

## Уредување
- Податоци за ценовник: `src/data/pricing.json`
- Текст „За мене“: `src/pages/About.jsx`
- Контакт форма: во моментов е демо (alert). Поврзи со email сервис (на пр. Formspree) или backend.
